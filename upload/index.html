Frequency Generator

Directions:

FreeGen is a small java app for generating, and panning between, frequencies. Its use is fairly straight forward.

  • To create a frequency generator, select the generator type from the "New" menu.
  • To remove a frequency generator, right-click and select "Remove".

Once you have active frequency generators:

Pure Tone

  • As the name suggests, you only have one frequency, "Freq 1". It can be adjusted with the keyboard and is between 10hz and 25,000hz.
  • The Play/Stop button toggles the state of the generator.
  • Adjusting the "Vol." slider adjusts the volume.

Panning

In addition to the settings allow for Pure Tones, Panning generators also have:

  • "Freq2" is the second frequency in the pan. It is not necessarily required to be greater than "Freq1".
  • The period is the time to complete a cycle in seconds.

Alternative Usage:

In the case that, for some reason, the applet will not load you have two options:

Option 1: Java Web Start

Java Web Start will take care of the downloading and running of a full, application version. To give it a try, select the launch link that follows.

Option 2: Executable Jar

You can download an executable jar with has both the stand-alone app, the applet, and the source: FreeGen.jar.

To execute the jar, first try just double clicking on it. If it launches, then you are golden. If it doesn't launch, you can run "java -jar FreeGen.jar" at the command line.
